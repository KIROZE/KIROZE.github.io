**v 0.2**

- 重构移动端界面
- 添加：移动端首页大图
- 添加：网站介绍闪烁跳动
- 添加：看板娘
- 添加：两款社交按钮（微信+steam）
- 添加：4款代码块主题风格
- 添加：友链模块
- 调整：分页按钮布局（虽然可能还有点Bug）
- 调整：社交按钮（并新增微信+steam）
- 修复：归档页无分页按钮bug
- 修复：移动端无评论界面的bug
- 修复：代码块自适应问题
- 修复：移动端无法加载valine评论系统

**v 0.1**

- 添加了博客自定义配置背景大图
- 侧边栏伸缩滑动
- 无封面图采用自定义配置图片
- 4款社交按钮
- 头像悬浮 旋转功能
- 暗黑金属系分页按钮-显示当前页面数
- 站点运行时间统计
- 浏览文章时加载目录树
- 浏览器滑动条美化（chrome等）
- 优先采用valine评论系统
- 标签云标签随机颜色显示
- 站内文章标题搜索功能
